# 发票合并打印工具 📄

一个纯前端的网页小工具，把多张「半张 A4 大小」的 PDF 发票**两两上下拼接成一张标准 A4**打印，省下一半纸张。

> 解决痛点：很多电子发票只有半张 A4 大小，单独打印浪费一半纸。这个工具把它们合并打印，既环保又高效。

---

## ✨ 功能特性

- 📎 **多文件上传**：支持点击选择，也支持直接拖拽 PDF 进来（可分多次累加拖入）
- 🗑️ **灵活管理**：每张发票可单独删除，也可一键清空全部
- 📐 **A4 精确排版**：使用 `@media print` 严格按 A4 尺寸排版，一页放 2 张发票，上下平分
- 📏 **智能缩放**：所有发票统一缩放到 19.3cm 宽（A4 可打印宽度），高度按原比例计算
- 🧠 **自动判断**：遇到超过半张 A4 的大发票，自动单独占一整页，不会被裁切
- 🔢 **奇数留白**：发票数为奇数时，最后一页下半部分自动留白
- 🖨️ **打印干净**：打印时自动隐藏所有操作界面，只显示发票内容
- 🔌 **开箱即用**：单个 HTML 文件，无需服务器，双击即用（仅首次加载 pdf.js 需联网）

---

## 🚀 使用方法

1. 下载本仓库的 `发票合并打印.html` 文件
2. **双击用浏览器打开**（推荐 Chrome / Edge）
3. 点击上传区域，或把 PDF 文件拖进虚线框
4. 在文件清单里确认/删除不需要的发票
5. 点击「预览并打印」按钮，下方会显示拼接后的 A4 预览
6. 按 `Ctrl + P`（或浏览器打印菜单），选择 A4 纸打印即可

> 打印预览时，操作界面会自动隐藏，只剩发票内容。

## 🧰 技术栈

| 技术 | 用途 |
|---|---|
| HTML + CSS + 原生 JavaScript | 主体实现，单文件，无构建步骤 |
| [pdf.js](https://mozilla.github.io/pdf.js/)（CDN） | 在浏览器里解析和渲染 PDF |
| `@media print` + `@page` | 精确控制 A4 打印排版与分页 |

### 核心处理逻辑

1. 用 pdf.js 读取每个 PDF 的每一页，渲染成图片
2. 把每页宽度缩放到 19.3cm，算出缩放后的高度
3. **配对规则**：
   - 高度 > 半张 A4（14.85cm）的发票 → 独占一整页
   - 其余发票按顺序两两上下拼到一页
   - 奇数张时最后一页下半留白
